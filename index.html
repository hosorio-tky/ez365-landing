Política de Privacidad — EZ365

Última actualización: 2026

1. Alcance

EZ365 es una aplicación utilizada por empleados autorizados de organizaciones que utilizan la plataforma EZ365. La aplicación requiere autenticación mediante usuario y contraseña.

2. Información que recopilamos

2.1 Datos de cuenta

  • Identificador de usuario.
  • Datos necesarios para autenticación y sesión.

2.2 Ubicación precisa (incluyendo en segundo plano)

  • La aplicación recopila ubicación GPS precisa del dispositivo.
  • La ubicación puede recopilarse incluso cuando la aplicación está cerrada o en segundo plano mientras el usuario tiene una sesión activa de trabajo (Clock In).
  • Se registran datos asociados como fecha, hora y precisión de la ubicación.

La ubicación se utiliza únicamente para:

  • Registrar recorridos de trabajo.
  • Verificar presencia en sitios de proyectos asignados.
  • Gestión operativa de actividades laborales.

2.3 Cámara y fotos

  • Fotos y videos capturados por el usuario dentro de la aplicación.
  • Las imágenes se utilizan para documentación de actividades en proyectos.
  • La aplicación no accede automáticamente a la galería del dispositivo sin acción del usuario.

2.4 Micrófono

  • Grabaciones de audio creadas por el usuario para notas de trabajo o evidencia en campo.

2.5 Archivos del dispositivo

  • Acceso a almacenamiento del dispositivo únicamente para guardar o adjuntar contenido generado dentro de la aplicación.

2.6 Notificaciones

  • Token de notificación push del dispositivo.
  • Mensajes relacionados con proyectos, tareas y avisos operativos.

2.7 Datos técnicos

  • Modelo de dispositivo.
  • Sistema operativo.
  • Versión de la aplicación.
  • Logs técnicos para diagnóstico.

3. Cómo usamos la información

  • Permitir el acceso a la aplicación.
  • Registrar sesiones de trabajo (Clock In / Clock Out).
  • Monitorear recorridos laborales durante la jornada.
  • Documentar actividades en proyectos.
  • Enviar notificaciones operativas.
  • Mejorar estabilidad y seguridad del sistema.

4. Uso de ubicación en segundo plano

EZ365 recopila ubicación en segundo plano únicamente cuando el usuario tiene una sesión de trabajo activa.

El seguimiento se detiene automáticamente cuando el usuario realiza Clock Out.

La ubicación no se utiliza para publicidad ni para seguimiento personal fuera del contexto laboral.

5. Compartición de información

No vendemos datos personales.

La información puede compartirse con:

  • La empresa empleadora del usuario para la gestión de proyectos y actividades laborales.
  • Proveedores tecnológicos necesarios para operar la aplicación (infraestructura, notificaciones).
  • Autoridades cuando la ley lo requiera.

6. Retención de datos

La información se conserva únicamente durante el tiempo necesario para la operación del servicio y según políticas de la organización que utiliza EZ365.

7. Seguridad

Utilizamos medidas de seguridad técnicas y organizativas para proteger la información, incluyendo conexiones seguras (HTTPS) y controles de acceso.

8. Permisos del dispositivo

La aplicación puede solicitar los siguientes permisos del dispositivo:

  • Ubicación precisa.
  • Ubicación en segundo plano.
  • Micrófono.
  • Almacenamiento del dispositivo.
  • Notificaciones.

Los permisos pueden ser administrados desde la configuración del dispositivo.

9. Menores de edad

La aplicación no está destinada a menores de edad.
